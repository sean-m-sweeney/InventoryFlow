1 Enable Custom App Development

First, you need to enable custom app development in your Shopify store if it isn't already enabled.

  1. Log in to your Shopify Admin panel
  2. Click Settings (gear icon) in the bottom-left corner
  3. Click Apps in the left sidebar
  4. Scroll down and click App and sales channel settings
  5. Click Develop apps
  6. If prompted, click Allow custom app development
  7. Read the warning and click Allow custom app development to confirm
‚ö†Ô∏è Store Owner Required

Only the store owner can enable custom app development. If you're a staff member, ask the store owner to complete this step or grant you the necessary permissions.

2 Create a Custom App

Now create a dedicated app for InventoryFlow to connect securely.

  1. From the Develop apps page, click Create an app
  2. Enter an app name: InventoryFlow (or any name you prefer)
  3. Select the app developer (usually yourself)
  4. Click Create app

3 Configure API Permissions

Grant the app permission to read your product and inventory data.

  1. In your new app, click Configure Admin API scopes
  2. Search for and enable the following scopes:
Scope Permission Purpose
read_products Read products View product names, images, SKUs
read_inventory Read inventory View stock levels at all locations
  1. Click Save to confirm the permissions
üí° Read-Only Access

InventoryFlow only needs read access. It cannot modify your products or inventory, keeping your store data safe.

4 Install the App & Get Your Token

Install the app to your store and retrieve your access credentials.

  1. Click the API credentials tab
  2. In the "Access tokens" section, click Install app
  3. Review the permissions and click Install
  4. Click Reveal token once to display your Admin API access token
  5. Copy the token immediately ‚Äî it starts with shpat_
üîê Important: Save Your Token Now

Shopify only shows the access token once. Copy it immediately and paste it into InventoryFlow. If you lose it, you'll need to uninstall and reinstall the app to generate a new token.

5 Connect InventoryFlow

Enter your credentials in the InventoryFlow application.

  1. Open InventoryFlow
  2. On the setup screen, enter:
    • Store Domain: Your store's URL (e.g., my-store.myshopify.com)
    • Admin API Access Token: The token you copied (starts with shpat_)
  3. Create a 4-digit PIN to secure the application
  4. Confirm your PIN and click Complete Setup
‚úÖ You're All Set!

InventoryFlow will connect to your store and begin syncing your inventory. Use the Sync button anytime to refresh the data.

6 Finding Your Store Domain

Not sure what your store domain is? Here's how to find it:

  • Look at the URL when logged into Shopify Admin: https://your-store.myshopify.com/admin
  • Or go to Settings ‚Üí Domains to see your myshopify.com domain

Enter just the domain part: your-store.myshopify.com

You can also enter just your-store and InventoryFlow will add the rest automatically.

Troubleshooting

‚ùå "API request failed" Error

  • Verify your store domain is spelled correctly
  • Make sure the access token was copied completely (no missing characters)
  • Check that the app is installed (not just created) in Shopify

‚ùå "Invalid credentials" Error

  • Regenerate your access token by uninstalling and reinstalling the custom app
  • Ensure you have the required API scopes enabled

‚ùå Products Not Showing

  • Verify your products have SKUs assigned
  • Check that products are set to "Active" status in Shopify
  • Click the Sync button to refresh data

‚ùå Forgot Your PIN

Delete the ~/.inventoryflow/inventoryflow.db file (in your home folder) and restart the app. You'll need to re-enter your Shopify credentials.

Security Information

InventoryFlow takes security seriously:

  • Encrypted Storage: Your Shopify token is encrypted using AES-256-GCM before being stored locally
  • PIN Protection: Your PIN is hashed using SHA-256 ‚Äî it's never stored in plain text
  • Local Data: All credentials are stored locally on your computer, never transmitted to third parties
  • Read-Only Access: The app only requests read permissions ‚Äî it cannot modify your Shopify data

For enhanced security in production environments, set the INVENTORYFLOW_SECRET environment variable to a custom encryption key.
